# MVP仕様

## 決定事項

- Windows専用
- 日本語UIのみ
- Tauri + React + TypeScript
- 対象は画像のみ
- 対象形式は `jpg`, `jpeg`, `png`, `webp`
- `heic` と動画は後回し
- 削除は完全削除ではなく、OSのゴミ箱へ移動
- 類似画像は自動選別せず、ユーザーが横並びで比較する
- スキャン結果は保存しない

## 画面

1. フォルダ選択
2. スキャン設定
3. 検出結果
4. 横並び比較
5. ゴミ箱へ移動する前の確認

## 検出

- 完全重複: SHA-256でファイル内容を比較
- 類似画像: 画像の見た目からpHashを作り、ハミング距離で比較
- ブレ画像: グレースケール画像のラプラシアン分散に近い値で鮮明度を推定
