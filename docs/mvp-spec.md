# MVP仕様

## 決定事項

- Windows専用
- 日本語UIのみ
- Tauri + React + TypeScript
- 対象は画像のみ
- 対象形式は `jpg`, `jpeg`, `png`, `webp`
- `heic` と動画は後回し
- 削除は完全削除ではなく、OSのゴミ箱へ移動
- 類似画像は自動選別せず、ユーザーが横並びで比較する
- スキャン結果は保存しない

## 画面

1. フォルダ選択
2. スキャン設定
3. 検出結果
4. 横並び比較
5. ゴミ箱へ移動する前の確認

## 検出

- 完全重複: SHA-256でファイル内容を比較
- 類似画像: 画像の見た目からpHashを作り、ハミング距離で比較
- ブレ画像: グレースケール画像のラプラシアン分散に近い値で鮮明度を推定

## 後回し

- HEIC対応
- ベストショット自動提案
- クラウド写真連携
- 多言語対応

## 大量スキャン対応

- スキャン対象フォルダ直下に `.picture-cleaner-cache.tsv` を作成する
- ファイルサイズと更新日時が変わっていない画像は再解析しない
- キャンセル時も確認済みのキャッシュを保存し、次回スキャンで再利用する
- 類似画像は総当たり比較ではなく、64bitハッシュを4つの16bitバケットに分けて候補を絞る
- 結果画面は一度に全件表示せず、25グループずつページングする

将来的に高度な検索、複数フォルダ横断、スキャン履歴管理が必要になったら、キャッシュ形式をSQLiteへ移行する。
